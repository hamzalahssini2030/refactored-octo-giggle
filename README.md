# Loria Space Prompt Engineer

A smart multilingual HTML tool for creating structured AI prompts quickly and clearly.

## Features

* Multilingual interface: English, Arabic, French, Simplified Chinese, and Hindi.
* Smart Fill system that fills prompt fields based on the selected language.
* Prompt export options.
* Clean interactive design.
* Works as a single HTML file.
* Ready to publish with GitHub Pages.

## How to use

1. Open the website.
2. Choose your preferred language.
3. Select the type of output you want to create.
4. Click Smart Fill or complete the fields manually.
5. Generate your final prompt.
6. Copy or export the result.
