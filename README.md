Loria Space Prompt Engineer

Loria Space Prompt Engineer is a single-file HTML tool designed to help users write clear, structured, and professional AI prompts without needing programming knowledge.

This project works directly in the browser. The user can open the HTML file, choose a language, select the type of output they want, fill in the fields manually or use the Smart Fill option, then generate a complete prompt ready to copy, export, or use with any AI platform.

What this project is

This is not a complex software installation or a backend system.It is a ready-to-use interactive HTML file that can run locally on any computer or be published online using GitHub Pages.

The main purpose of this tool is to make prompt writing easier, faster, and more organized for users who want to create:

Text prompts

Images

Videos

Documents

Code projects

Websites

Applications

Dashboards

AI workflows

Educational or professional content

Main features

Works as one single index.html file.

Can be downloaded and opened locally in any modern browser.

Can be shared with anyone as a file.

Can be published online using GitHub Pages.

Does not require installation.

Does not require a server.

Does not require a database.

Includes a multilingual interface.

Includes Smart Fill to help users complete prompt fields quickly.

Helps users generate structured prompts for different AI tools.

How to use it locally

Download the index.html file.

Open it with any modern browser such as Chrome, Edge, Firefox, or Safari.

Choose your preferred language.

Select what you want to create.

Fill in the fields or use Smart Fill.

Generate the final prompt.

Copy or export the result.

How to publish it online

This project can be hosted for free using GitHub Pages:

Upload the index.html file to a GitHub repository.

Go to repository Settings.

Open Pages.

Choose the main branch as the source.

Save the settings.

GitHub will generate a public website link.
